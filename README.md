- 트립트룹(TRIPTROOP)은 “여행 그룹” 을 의미하며 여행을 함께할 사람을 모으고 여행 정보를 공유하는 서비스 입니다.
- 순수한 정보 공유와 커뮤니티 활성화에 중점을 두어 여행자들이 보다 풍부하고 진정한 여행 경험을 할 수 있도록 돕습니다.
